# Rainfall Prediction System

## Project Description
This project predicts rainfall using Machine Learning and Flask.

## Important Note
Model (.pkl) files are not included due to size limitations.

## How to Run the Project

1. Open the Jupyter Notebook file.
2. Run all cells to generate:
   - rainfall.pkl
   - encoder.pkl
   - imputer.pkl
   - scale.pkl
3. After model files are created,  Then add thise files into
   your flask project folder where place under app.py.
4.Now run the code :
python app.py

5. Open browser and go to:
http://127.0.0.1:5000/
